# 标注系统快捷键指南

## 🗺️ 导航与视图
| 按键 | 功能描述 |
| :--- | :--- |
| **A** | 上一张图片 |
| **D** | 下一张图片 |
| **鼠标滚轮** | 缩放图片 |
| **鼠标右键拖拽** | 平移画布 |
| **Ctrl + Shift + D** | 删除当前图片（需确认） |

## 🛠️ 模式切换
| 按键 | 功能描述 |
| :--- | :--- |
| **W** | 切换到 **普通标注模式** (Annotate) |
| **Q** | 切换到 **编辑/选择模式** (Edit/Select) |
| **~** 或 **`** | 切换到 **AI 辅助模式** (SAM) |

## 📝 标注编辑 (通用)
| 按键 | 功能描述 |
| :--- | :--- |
| **1** - **9** | 快速切换标签类别 (对应侧边栏顺序) |
| **Tab** | 选中下一个标签类别 |
| **Shift + Tab** | 选中上一个标签类别 |
| **Del** 或 **E** | 删除选中的标注框 |
| **K** | **清空当前图片上的所有标注** (需确认) |
| **Esc** | 取消当前绘制 / 取消选中 / 关闭菜单 |
| **Ctrl + Z** | 撤销上一步操作 |
| **Ctrl + C** | **复制** 选中的矩形框 |
| **Ctrl + V** | **粘贴** 矩形框 (以鼠标位置为中心) |
| **方向键 (↑↓←→)** | 微调选中框的位置 |
| **Ctrl + 方向键** | 大幅度调整选中框的位置 |

## 🤖 AI 辅助模式 (SAM)
*(当处于 AI 辅助模式时有效)*

| 按键 | 功能描述 |
| :--- | :--- |
| **Space (空格)** | **开始推理**：将当前的提示点/框发送给 AI 生成轮廓 |
| **R** | 切换 AI 模式：**提示词关联 (Semantic)** ↔ **单体分割 (Standard)** |
| **Shift + 拖拽** | 绘制 **红色** 排除框 (Negative Prompt)，告诉 AI "不是这里" |
| **Esc** | 清除当前的辅助提示点/框 |

## 🖱️ 鼠标操作
- **左键单击**：绘制点 / 选择框。
- **左键拖拽**：绘制矩形 / 调整框大小。
- **右键单击**：(多边形模式) 闭合多边形 / (普通模式) 打开右键菜单。
- **右键拖拽**：平移画布。
