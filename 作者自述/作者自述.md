# 🚀 项目名称：[YOLOv8-YOLO26-Labeling-and-Training-Tool]
### 一个由“生气”驱动的视觉标注全生命周期工具

## 📖 项目背景
我是一名刚本科毕业、从事视觉方向工作的开发者。在日常工作中，我尝试过市面上大部分开源标注工具，但总会遇到各种痛点：要么安装环境极其复杂，要么在“标注-训练-导出”的流程衔接上非常繁琐。

在某次因工具导出格式崩溃而“生气”后，我决定自己从零写一个。

## 💻 技术特点
*   **自研核心**：本项目约 1.2 万行代码，完全由我和 AI 协同完成。为了保证底层逻辑的可控性，**未采用任何第三方现成框架**。
*   **紧跟前沿**：在项目成型之初，正值 SAM3 (Segment Anything) 更新，我在拿到模型后的当周内即完成了集成。
*   **实战验证**：项目经过了 3 个月的实际生产环境测试，目前已知的 Bug 均已修复，稳定性足以应对日常任务。
*   **全流程闭环**：支持从**标注**到**训练**再到**导出**的完整生命周期，旨在解决视觉工作者最头疼的流程打通问题。

## 🎁 开源范围
本次开源的版本可以满足绝大多数常规视觉任务的需求：
- [x] 核心标注功能
- [x] 模型训练接口
- [x] 多格式数据导出
- [x] 适配最新的 YOLO26 系列（如 YOLOv8/v10/v11/26 等）

### ⚠️ 注意事项（当前版本暂不包含）
目前有约 3200 行代码涉及的高级功能暂未开源，包括：
*   OBB（旋转框）标注与矩形框标注
*   全自动/半自动辅助标注
*   语义提示词（Text Prompt）标注

如果你对视频展示中的上述高级功能有需求，可以联系我获取相关服务支持。
